# ACLS Shareholder Letters — Pattern C (Annual Report front matter)

**Pattern classification**: **C** — no standalone CEO shareholder letter.

Axcelis does NOT publish a standalone annual shareholder letter (no Buffett-style or Tobi-Lütke-style letter). The closest analog is the **Chairman/CEO Letter contained in the Annual Report (10-K wrapper)** mailed to stockholders alongside the proxy.

## Gap acknowledgment

Per Rule #19 Pattern C: when a standalone letter does not exist, fall back to (a) AR front-matter chairman's letter, (b) DEF 14A introductory letter, and (c) earnings call transcripts. ACLS provides limited front-matter narrative and typically only at year-end. Earnings transcripts (especially Q4 wrap calls) carry the bulk of management's strategic framing for ACLS — these are stored in `raw/ACLS/transcripts/` and synthesized in §6 of the wiki page.

5-year multi-letter arc table is therefore replaced by a 2–4 sentence synthesis paragraph in §6 RMC subsection (per Rule #21 — synthesis over transcription).

This file documents the Pattern C classification so the gap is auditable.
