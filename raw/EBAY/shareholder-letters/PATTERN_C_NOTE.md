# EBAY — Shareholder Letter Pattern (Core Rule #19)

**Pattern: C** — No standalone annual CEO/founder shareholder letter.

eBay does not publish a Buffett-style annual letter, nor does it publish a
DASH/SHOP-style quarterly letter to shareholders. CEO Jamie Iannone's
formal management commentary surfaces in three primary-source channels:

1. **Quarterly earnings press releases** — Iannone "shareholder section"
   commentary at the top of each release (https://investors.ebayinc.com/financial-information/quarterly-results/)
2. **Quarterly earnings call transcripts** — opening prepared remarks
3. **Annual proxy statement (DEF 14A)** — chairman / CEO introductory letter
   in front matter (filed each spring)

For first-run synthesis under v2.6+ retrofit:
- Section 6 RMC pulls from the most recent quarterly PR commentary +
  DEF 14A intro letter as the closest available primary-source proxies.
- A multi-year strategic-framework arc is constructed from earnings-PR
  CEO commentary across FY2021–FY2025.

No raw letter PDFs were downloaded for EBAY because no such standalone
artifact exists. Gap logged here per Core Rule #19 Pattern C guidance.
