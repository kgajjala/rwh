# RH — Shareholder Letters

## Publication Pattern

**Pattern C (transitional)**: RH published standalone annual shareholder letters for FY2019–FY2021 only.
Starting FY2022, no standalone annual shareholder letter has been published — Gary Friedman
communicates strategy via: (1) Q4 video presentations posted on ir.rh.com, (2) earnings call commentary,
and (3) 10-K MD&A forward-looking statements language.
