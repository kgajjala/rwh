# SharkNinja Shareholder Letter Pattern Note

**Date assessed**: 2026-04-26
**Determination**: **Pattern C — No standalone CEO shareholder letter**

SharkNinja, post its July 2023 spin-off from JS Global, does NOT publish a
standalone annual letter to shareholders. CEO Mark Barrocas does not file a
"Letter to Shareholders" parallel to a Buffett or a Tobi Lütke. Instead,
management commentary lives in:

1. **Quarterly earnings press releases** (Mark Barrocas top-of-release quotes
   serve as the de facto management commentary; Q4/FY release each year is the
   closest analog to an annual wrap)
2. **Quarterly earnings call transcripts** (Barrocas + CFO Adam Quigley
   prepared remarks)
3. **20-F Item 4/5 (Operating & Financial Review) front matter** — SN is a
   Cayman-incorporated foreign private issuer and files 20-F annual reports,
   not 10-K. The 20-F includes a strategic overview but not a personal
   letter format.

**Sub-5-year coverage** (per Core Rule #19): SN has only been public since
July 2023. Available public-company annual cycles:
- FY2023 (partial year as public; 20-F filed early 2024)
- FY2024 (20-F filed March 2025)
- FY2025 (Q4 release Feb 11 2026; 20-F expected late March 2026)

**Synthesis approach** (per Pattern C fallback): pull verbatim Barrocas /
Quigley quotes from the most recent earnings calls and press releases.
The Q4 2025 earnings call (2026-02-11) is the highest-fidelity recent
management framework articulation and is treated as the de facto FY2025
annual letter equivalent.
