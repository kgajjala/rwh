# Asset Type Adaptations

How the 15-section framework adapts by business model type.

| Type | Example | Key Volume Metric | Valuation Primary | Moat Source |
|------|---------|-------------------|-------------------|-------------|
| Capital-light platform | BKNG | GBV, Room Nights, Take Rate | FCF yield, EV/EBITDA | Network effects, switching costs |
| Franchise royalty | WING | SSS, AUV, Unit Count | EV/EBITDA | Brand, unit economics |
| Financial/Brokerage | SCHW | NII, NIM, AUM, Net New Assets | P/TBV, P/E | Switching costs (custody), scale |
| Pharma/Biotech | LLY | Revenue by drug, pipeline milestones | P/E, EV/Sales | IP/patents, R&D scale |
| Managed Care | UNH | MLR, membership, premium yield | P/E, P/FCF | Scale, vertical integration |

See CLAUDE.md for full per-type guidance on each section.
